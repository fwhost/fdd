Summary:DotDash Keyboard
Description:
A keyboard using non-standard "untimed" Morse code. Tap out the code for a letter and hit enter to display it. Don't worry if you don't know it: a cheat-sheet is accessible by swiping up from the keyboard. The utility keyboard, accessible by swiping to the side must be enabled in the settings for it to become available.
